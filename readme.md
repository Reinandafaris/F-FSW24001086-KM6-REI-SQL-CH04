## API Reference

#### Get default router

```http
  GET http://localhost:3000/
```

#### Get create listcar page

```http
  GET /api/v1/listcars/create
```

#### Get all listcars

```http
  GET /api/v1/listcars/all
```

#### Create car

```http
  POST /api/v1/listcars/admin/create
```

#### Update car

```http
  POST /api/v1/listcars/admin/edit/:id
```

#### Delete car

```http
  POST /api/v1/listcars/delete/:id
```
